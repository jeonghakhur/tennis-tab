# Plan: 테니스 레슨 슬롯 예약 시스템 (Lesson Slot Booking)

> 작성일: 2026-03-16
> 목적: 기존 레슨 프로그램 방식에서 **슬롯 기반 예약 방식**으로 전환
> 참고: `tennis-lesson.plan.md` (기존 플랜)

---

## 개요

코치가 레슨 가능한 시간 슬롯을 30분 단위로 미리 등록해두고,
회원이 빈 슬롯을 직접 선택해서 신청하는 방식.

코치가 특정 슬롯에 레슨자를 직접 배정하면 해당 슬롯은 잠기고,
나머지 빈 슬롯만 회원이 선택할 수 있다.

---

## 핵심 개념

```
코치 (coaches)
  └── 레슨 슬롯 (lesson_slots) ← 코치 또는 어드민이 미리 등록
        ├── OPEN     : 빈 슬롯 (회원 신청 가능)
        ├── LOCKED   : 코치/어드민이 특정 회원 직접 배정 (잠김)
        └── BOOKED   : 회원이 신청 완료 (잠김)

회원 (club_members)
  └── 레슨 예약 (lesson_bookings) ← 슬롯 선택 신청
        ├── 주 1회 : 슬롯 1개 선택
        └── 주 2회 : 슬롯 2개 선택 (주중/주말 혼합 가능)
```

| 액터 | 역할 |
|------|------|
| 클럽 어드민 | 슬롯 생성/삭제, 코치별 슬롯 관리, 회원 직접 배정 |
| 코치 (어드민 대리) | 어드민이 코치 대신 슬롯 및 배정 관리 |
| 클럽 회원 | 빈 슬롯 1~2개 선택 후 신청 |

## 레슨 가능 시간 (기본 정책)

| 요일 | 레슨 가능 시간 |
|------|--------------|
| 월, 화, 목 | 06:30 ~ 19:30 |
| 수, 금 | 06:30 ~ 14:00 |
| 토, 일 | 06:30 ~ 10:00 |

> 슬롯 등록 시 위 시간 범위 내에서만 등록 가능하도록 유효성 검사 적용.
> 코치별로 별도 가능 시간 설정도 가능하게 추후 확장 예정.

## 슬롯 등록 방식

### 코치/어드민이 슬롯 생성

- 날짜 + 시작 시간 + 30분 단위 자동 계산
- 반복 패턴 지원: 매주 특정 요일 반복 (예: 매주 화, 목)
- 슬롯 생성 시 기본 상태: `OPEN`

```
예시: 매주 화/목 10:00~12:00 등록 시
→ 10:00~10:30 OPEN
→ 10:30~11:00 OPEN
→ 11:00~11:30 OPEN
→ 11:30~12:00 OPEN
(4주치 자동 생성)
```

---

## 슬롯 상태

| 상태 | 설명 | 신청 가능 여부 |
|------|------|--------------|
| `OPEN` | 빈 슬롯 (회원 신청 가능) | ✅ 가능 |
| `BLOCKED` | 코치가 비공개 처리 (레슨 없음, 개인 사정 등) | ❌ 불가 |
| `LOCKED` | 코치/어드민이 특정 회원 직접 배정 | ❌ 불가 |
| `BOOKED` | 회원이 신청 완료 | ❌ 불가 |
| `CANCELLED` | 취소됨 (기존 예약 취소) | ❌ 불가 |

> **BLOCKED vs CANCELLED 차이:**
> - `BLOCKED`: 슬롯 자체를 열지 않음 (코치 휴무, 개인 사정 등) — 회원에게 노출 안 됨
> - `CANCELLED`: 기존에 OPEN/BOOKED 상태였다가 취소된 것 — 회원에게 취소 알림 발송

---

## 요금 설정

### 슬롯 수 × 주중/주말 조합별 요금

회원이 선택한 슬롯 수와 요일 조합에 따라 요금 자동 계산:

| 선택 | 요금 기준 |
|------|---------|
| 주 1회 (주중) | `fee_weekday_1` |
| 주 1회 (주말) | `fee_weekend_1` |
| 주 2회 (주중 2개) | `fee_weekday_2` |
| 주 2회 (주말 2개) | `fee_weekend_2` |
| 주 2회 (주중 1 + 주말 1) | `fee_mixed_2` |

> 요금은 **레슨 프로그램**에 설정하며, 슬롯 선택 시 자동 계산되어 표시됨.

### 요금 설정 규칙

- 최대 **2개 슬롯** 선택 가능
- 주중/주말 혼합 선택 가능
- 슬롯 선택 시 실시간으로 예상 요금 표시
- 월 단위 요금 (월 4회 기준 주 1회, 월 8회 기준 주 2회)

---

## 회원 신청 플로우

```
1. 레슨 신청 화면 진입
   └── 코치 선택 → 해당 코치의 빈 슬롯 달력 표시

2. 슬롯 선택 (최대 2개)
   ├── 1개 선택 → 주 1회 요금 표시
   └── 2개 선택 → 주 2회 요금 표시 (주중/주말 혼합 가능)

3. 요금 확인 → 신청 완료
   └── 선택한 슬롯 상태 OPEN → BOOKED 변경
   └── 나머지 같은 시간대 슬롯 자동 OPEN 유지

4. 신청 확정 알림 발송 (회원)
```

---

## 코치 배정 플로우 (어드민)

```
1. 어드민 → 슬롯 목록에서 특정 슬롯 선택
2. 회원 검색 → 배정
3. 슬롯 상태 OPEN → LOCKED 변경
4. 배정된 회원에게 알림 발송
```

## 데이터 모델

### `lesson_slots` 테이블 (신규)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| program_id | uuid | FK → lesson_programs |
| coach_id | uuid | FK → coaches |
| slot_date | date | 레슨 날짜 |
| start_time | time | 시작 시간 |
| end_time | time | 종료 시간 (start + 30분) |
| day_type | enum | `WEEKDAY` / `WEEKEND` |
| status | enum | `OPEN` / `LOCKED` / `BOOKED` / `CANCELLED` |
| locked_member_id | uuid | FK → club_members (LOCKED 시) |
| created_by | uuid | FK → profiles |
| created_at | timestamptz | |
| updated_at | timestamptz | |

### `lesson_bookings` 테이블 (신규)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| member_id | uuid | FK → club_members |

| fee_amount | int | 적용 요금 (원) |
| status | enum | `PENDING` / `CONFIRMED` / `CANCELLED` |
| confirmed_at | timestamptz | |
| cancelled_at | timestamptz | |
| created_at | timestamptz | |

## 기능 요구사항

### 슬롯 관리 (어드민)

| ID | 요구사항 |
|----|---------|
| FR-01 | 어드민이 코치별 슬롯을 날짜/시간으로 등록 |
| FR-02 | 반복 패턴 지원 (매주 특정 요일, N주치 일괄 생성) |
| FR-03 | 슬롯 상태 변경 (OPEN ↔ BLOCKED) — 레슨 없는 시간 비공개 처리 |
| FR-04 | 특정 슬롯에 회원 직접 배정 (OPEN → LOCKED) |
| FR-05 | 달력 뷰로 슬롯 현황 확인 |

### 슬롯 신청 (회원)

| ID | 요구사항 |
|----|---------|
| FR-10 | 코치 선택 후 해당 코치의 빈 슬롯만 표시 |
| FR-11 | 슬롯 최대 2개 선택 (주중/주말 혼합 가능) |
| FR-12 | 슬롯 선택 시 실시간 요금 계산 및 표시 |
| FR-13 | 신청 완료 시 슬롯 상태 BOOKED 변경 |
| FR-14 | 신청 취소 시 슬롯 상태 OPEN으로 복구 |
| FR-15 | 내 예약 현황 조회 |

### 요금 계산

| ID | 요구사항 |
|----|---------|
| FR-20 | 슬롯 1개 선택 → 주중/주말 구분하여 요금 표시 |
| FR-21 | 슬롯 2개 선택 → 주중2/주말2/혼합 구분하여 요금 표시 |
| FR-22 | 요금 미설정 시 "요금 문의" 표시 |
