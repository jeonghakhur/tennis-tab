# 테니스 대회 생성 및 대회 참가, 대회 신청 등록하는 웹 서비스를 개발하려해

대회 관련 정보를 보여주는 포스트 기능도 작성하고 작성한 좋아요와 댓글 기능도 개발하려구해
회원가입을하면 클럽을 생성할 수도 있고 회원들을 등록할 수 있는 기능도 있어
서비스 PRD 문서를 docs/PRD.md 파일로 생성해줘

현재 서비스는 사용자가 자연어를 입력해서 경기 결과도 등록하고 모든 서비스를 자연어로 처리해 볼 수 있게 구성하고 싶은데 어떤 처리 과정이 필요하지

로그인 전에 LLM 서비스를 통해 내가 할 수 있는 일을 알려주고 싶어

1. 특정 키워드 대회 검색
2. 내가 참가신청한 대회 검색
3. 내가 참가 신청한 대회에 참가 신청이 완료되었는지.
4. 대진표 검색
5. 결과 검색
6. 참가 기준 검색.
7. 나의 경기 대회 결과 등록
8. 나의 대회 장소 및 코트 번호 검색
우선 내가 생각나는 부분은 이정도 이고 대회 생성. 대진표 자동 등록등은 관리자만 할 수 있게 처리할꺼야
이런 기준으로 자연어로 처리하는 것으로 다시 정리해주고 어떤 LLM서비스를 사용해야 할지도 검토해봐

사용자가 로그인을하면 로그인 버튼이 사라지고 로그인 사용자의 아바타르 표시하고 사용자의 프로필 이미지가 없나면 이니셜의 첫글짜가 나오는 형태로 수정해.
사용자의 아바타를 클릭하면 본인 마이페지로 접속 가능하게하고 마이페이지에는 나의 프로필 정보와 나의 참가신청 대회와 나의 경기 결과를 보여주고 추후 나를 팔로워하는 기능을 넣으면 나를 팔로워한 사람 목록과 내가 팔러워 한 사람을 보여주는 UI를 content7 MCP를 사용해 만들어 팔러워 한 사람의 아바타를 클릭하면 해당 유저의 게임 결과를 볼 수 있게 처리할꺼야~
디자인은 현재 톤앤배너를 참고하고 프론트앤드 디자인 스클을 계속 사용해

로그인 후 페이지 새로고침시 로그인 버튼이 깜빡이게 보여지고 있는데 이 부분 깜빡이지 않도록 수정해.

슈파베이스의 profiles 테이블에서 dominant_hand 필드를 삭제하고 클럽의 지역을 입력하는 필드를 추가해.
그리고 프로필 페이지에서 주 사용 손 필드를 삭제하고 클럽의 지역을 입력하는 필드를 추가해. 지역은 시도, 시군구 단계로 입력하게해. 그리고 너가 작성한 마이그레이션 파일을 수정해. 그리고 자동으로 입력 테스트도 진행해

구력 필드는 입문 년도 필드로 변경하고 현재 년도가 처음 나오고 현재 년도 부터 10년 전까지 년도출 선택할 수 있게 수정하고 10년 이상은 해당년도 이전으로 선택할 수 있게 수정하고 skill_level 필드의 값은 선택한 년도가 들어가게 변경하고 10년 이상을 선택하면 10년 이상이라는 값이 들어가게 해줘.
그리고 점수를 입력할 수 있는 필드를 제공하고 점수는 숫자만 입력가능하게 input type="text"에 inputmode="numeric" 속성을 추가해줘.
