🔨 구현 기능
✏️ 대진 개별 수정
각 경기 카드에 수정 버튼 추가, 모달에서 선수·코트·시간·점수 수정 가능
신규 기능
  • updateMatchResult(matchId, input) 서버 액션 추가 (session-actions.ts)
  • 선수1/선수2 드롭다운 (참석 멤버 목록), 코트 번호, 시간, 점수 입력 필드
  • 점수 입력 시 자동으로 승자 계산 + 상태 COMPLETED 전환
  • 관리자(isAdmin) 일 때만 ✏️ 버튼 노출
PASS ✏️ 클릭 → 수정 모달 오픈, 선수 드롭다운 정상 표시
PASS 코트 "1번", 점수 6:4 입력 후 저장 → 완료 상태로 변경 확인
PASS 서버에서 winner_member_id 자동 계산 (E2E 테스터 6점 > 최재원 4점)
🗑 대진 개별 삭제
각 경기 카드에 삭제 버튼 추가, 확인 다이얼로그 후 삭제
개선
  • 기존 deleteMatchResult를 MatchBoard에서도 직접 호출
  • 기존에는 BracketEditor(관리 페이지)에만 삭제 버튼 있었음 → 세션 상세 페이지에도 추가
  • ConfirmDialog("이 경기를 삭제하시겠습니까?") 후 처리
  • 완료(COMPLETED) 경기는 서버에서 삭제 거부
PASS 🗑 클릭 → "이 경기를 삭제하시겠습니까?" 다이얼로그 표시
PASS 확인 → 경기 즉시 제거, 목록 132경기 → 131경기로 감소 확인
🗑🗑 전체 대진 일괄 삭제
대진표 상단 "전체 삭제" 버튼, 2단 확인 후 세션 전체 대진 삭제
주요 기능
  • deleteAllMatchResults(sessionId) 서버 액션 추가
  • Supabase에서 session_id 기준 전체 DELETE 1쿼리로 처리
  • ConfirmDialog("모든 대진 데이터를 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.") 경고 메시지
  • 삭제 후 "모든 대진이 삭제되었습니다." 성공 Alert 표시
PASS 전체 삭제 클릭 → 경고 다이얼로그 표시
PASS 확인 → 131경기 전부 삭제, 대진표 "아직 대진이 편성되지 않았습니다" 표시
PASS 페이지 리프레시 후에도 데이터 없음 확인
